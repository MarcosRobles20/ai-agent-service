# AI Agent Service

Python microservice implementing the AI reasoning engine of the chatbot platform.

The agent is built using LangGraph and LangChain and can execute tools, retrieve information and generate responses using local LLMs.

---

# Capabilities

* conversational reasoning
* tool execution
* web search
* HTTP requests
* summarization
* knowledge retrieval (RAG)
* real time streaming responses

---

# Architecture Role

Frontend

↓

ASP.NET Backend

↓

AI Agent Service

The backend sends chat requests to this service and receives streaming responses generated by the agent.

---

# Technologies

* Python
* LangGraph
* LangChain
* SQLite
* Local LLMs

---

# Agent Architecture

The agent is implemented using a LangGraph workflow composed of:

* reasoning nodes
* tool execution nodes
* routing logic
* event streaming

This design allows the agent to dynamically decide when to call tools or generate a final response.
